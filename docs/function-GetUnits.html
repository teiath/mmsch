Μονάδες

Η συνάρτηση αυτή επιστρέφει Μονάδες σύμφωνα με τις παραμέτρους που έγινε η κλήση.
Η κλήση μπορεί να γίνει μέσω της παρακάτω διεύθυνσης με τη μέθοδο GET και route_api_name = units :
https://mm.sch.gr/api/units
